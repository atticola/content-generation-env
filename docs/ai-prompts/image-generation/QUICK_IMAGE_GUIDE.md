# Quick Image Generation Guide

**Fast reference for creating images for health content**

## âš¡ Quick Specs

### Dimensions
- **16:9** horizontal (e.g., 1280x720px)
- **Core zone**: Center 9:9 square
- **Main subject**: MUST be in center

### File
- **Format**: WebP or JPEG
- **Size**: 50-150KB
- **Quality**: High but optimized

### Style
- **NO TEXT**
- **NO faces**
- Simple, clean, medical illustration
- Soft colors, professional
